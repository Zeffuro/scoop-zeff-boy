# scoop-zeff-boy

Scoop bucket for [zeff-boy](https://github.com/Zeffuro/zeff-boy) — a Game Boy, Game Boy Color, and NES emulator written in Rust.

## Installation

```powershell
scoop bucket add zeff-boy https://github.com/Zeffuro/scoop-zeff-boy
scoop install zeff-boy
```

## Updating

```powershell
scoop update zeff-boy
```
